Easily add your location map. Simple with three different styles: Aqua, black or gray 

== Description ==

Easy Location Map allows you to easily add a google map and choose between different styles ( gray, aqua or black). It can be included in posts or pages like this: [google_map] or directly in a template with:
<?php echo do_shortcode("[google_map]"); ?>. You can also choose the marker icon for the map



== Installation ==

1. Upload the entire `easy_location_map` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Easy Location Map' menu in your WordPress admin panel.

For basic usage, you can have a look at https://amrani.es/wp-plugins/easy-map.php
